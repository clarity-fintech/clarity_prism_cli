## System summary

```
PRISM  → decides WHAT should happen  (intent, prediction, PoR, ledger)
HELIX  → decides HOW it should happen (routing, MEV shield, execution)
SKILLS → decides WHAT VALUE it creates (modular deterministic runtime)
```

Global CLI command: **`clrt`**
