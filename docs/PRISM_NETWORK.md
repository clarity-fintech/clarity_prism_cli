# PRISM Network (CPN)

Proof-of-Relevance (PoR) capital context network.

**API:** `POST /v1/prism/intent-aware` · `GET /v1/prism/status`

## Node types

Intent-Aware RPC, Predictive Query, Risk Oracle, Portfolio Index, Capital Context, Liquidity Scout, Yield Mapper, Attestation Bridge, Settlement Monitor, Treasury Feed, Market Pulse, Inference Relay.

## Tiers

| Tier | Access | PoR multiplier |
|------|--------|----------------|
| I Reader | read-only feeds | 1× |
| II Contributor | submit vectors | 2× |
| III Curator | relevance curation | 4× |
| IV Architect | CPN governance | 8× |
